# Pluto Food Diary Prompt

AI prompt for structured food journaling, calorie estimation, and eating pattern analysis.

Pluto is a conversational food diary prompt designed for ChatGPT.

It helps log food, estimate calories and macros, track hunger and satiety, 
and analyze eating patterns over multiple days.

## Features

- Food logging with time
- Automatic calorie and macro estimation
- Daily summary
- Export of food diary
- Reflection and pattern analysis

## Example

8:33 salad 70 g and olive oil  
9:53 full  
10:25 salad 500 g and cream 50 g
